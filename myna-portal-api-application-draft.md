### サービス概要（記入例）

```
サービス名: AllYourFit（オールユアフィット）

サービス概要:
個人の健康・フィットネスデータを統合管理するパーソナルヘルスインテリジェンスサービスです。
トレーニング記録・食事管理・身体データ（体重・体脂肪・血圧）に加え、
マイナポータルと連携することで健康診断結果・薬剤情報・予防接種記録を一元管理します。
AIによる健康リスク分析・改善提案・サプリメント推奨機能を提供します。

利用目的:
ユーザー本人の同意のもと、特定健診情報・薬剤情報・予防接種記録を取得し、
パーソナルトレーナーとの連携による健康改善プログラムの最適化に活用します。
医療診断を行うものではありません。

想定ユーザー:
健康意識の高い個人・パーソナルトレーニング利用者（20〜60代）

開発体制:
自社開発（Next.js / Supabase / Claude API）
本番環境: Vercel（サーバーレス）

個人情報の取扱い:
・取得データはSupabase（PostgreSQL）に暗号化して保存
・第三者への提供は行わない
・ユーザーの同意取得・撤回機能を実装
・PHRサービス提供者向け基本的指針に準拠する
```
