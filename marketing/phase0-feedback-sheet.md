# CF Phase 0 フィードバックシート
**用途: 最初の10人から収集するフィードバック**
**作成: 橘凜（マーケ）**
**使い方: このシートをコピーしてGoogle FormsかLINEで送付**

---

## 対象者への説明文

```
CFをお試しいただきありがとうございます！
正直な感想を聞かせてください。5分以内で終わります。
改善に直接反映します。
```

---

## 回収後のアクション（橘凜が実施）

| 回答 | アクション |
|------|-----------|
| 「面倒」「無理そう」が50%超 | スクショ送信フローの再設計をMioに依頼 |
| 「なるほど瞬間なし」が多い | オンボーディングのコーチングメッセージを強化 |
| 「紹介したい」が50%超 | リファラルプログラムをSprint 300で前倒し |
| 機能要望が集中した項目 | 次スプリントのバックログに追加 |
